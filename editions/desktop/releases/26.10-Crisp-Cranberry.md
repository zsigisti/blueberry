# Blueberry Desktop 26.10 — “Crisp Cranberry”

The October 2026 release of Blueberry Desktop. Named in the project's house
style: **adjective + berry variety, alliterative, advancing through the
alphabet** (24.04 *Bright Bilberry* → **26.10 Crisp Cranberry** → 27.04 *Dapper
Dewberry* …). See [`editions/desktop/codenames`](../codenames).

## At a glance

| | |
|---|---|
| **Version** | 26.10 |
| **Codename** | Crisp Cranberry |
| **Channel** | Standard (not LTS) |
| **Released** | October 2026 |
| **Supported until** | July 2027 (9 months) |
| **Default desktop** | KDE Plasma 6 (Wayland) · GNOME spin optional |
| **Kernel** | Linux 7.0 — pinned for the life of the release |
| **Installer** | Calamares (live ISO) |
| **Architectures** | x86_64 |

> Next LTS is **28.04** (April 2028). The previous LTS is **26.04 “Bright
> Bilberry”** (supported 24 months).

## Identity

- **Logo** — the Blueberry mark (two berries + leaves), [`assets/drawing.svg`](../../../assets/drawing.svg),
  rendered to `logo.png` for the installer and login.
- **Wordmark** — “Blueberry Desktop 26.10 (Crisp Cranberry)”.
- **Typeface** — DejaVu Sans (UI), DejaVu Sans Mono (terminal). Noto is the
  planned upgrade for full Unicode coverage.

## Theme & palette

The brand is a **blueberry indigo** that deepens to near-black and lifts to a
clear cornflower accent. These are the canonical values (from the Calamares
branding and slideshow):

| Token | Hex | Use |
|---|---|---|
| Berry Deep | `#2a1a4a` | Primary background / installer sidebar |
| Berry Mid | `#3d2a6b` | Gradient partner, panels |
| Berry Glow | `#5b8def` | **Accent** — selection, links, highlights |
| Ink | `#f4f1fb` | Primary text on dark |
| Sub | `#c9bfe6` | Muted/secondary text |

- **Plasma theme:** Breeze, with the Berry Glow accent.
- **Installer (Calamares):** indigo sidebar (`#2a1a4a`) → mid (`#3d2a6b`)
  gradient, white sidebar text, Berry Glow selection. See
  [`calamares/branding/blueberry/`](../calamares/branding/blueberry/).
- **Wallpaper:** the default Breeze “fluid ribbons” in blue/red; a Cranberry
  release wallpaper (deep-indigo field with a cranberry-red glow) is the
  intended per-release art — drop it in as `wallpaper.png` and point
  `kdeglobals` / the SDDM theme at it.

## Cutting this release

Build the ISO with the version + codename pinned (otherwise they derive from the
date — see [`release.mk`](../release.mk)):

```sh
make desktop-iso BBD_VERSION=26.10 BBD_CODENAME="Crisp Cranberry"
# → iso/blueberry-desktop-26.10-kde-x86_64.iso
```

The build substitutes `@@VERSION@@`, `@@FULLVERSION@@`, and `@@CODENAME@@` into
the Calamares branding and the live session.
